Ejecuta el script con curl
Para ejecutar el menú desde cualquier servidor o terminal:

bash

curl -sSL https://raw.githubusercontent.com/The-DiosBot-MD/i/main/run.sh | bash

O si prefieres descargarlo primero:

bash
curl -O https://raw.githubusercontent.com/The-DiosBot-MD/i/main/run.sh
chmod +x run.sh
./run.sh
